# 🌍 AI Language Translation Tool

This project was developed as part of the CodeAlpha Artificial Intelligence Internship.

## 📌 Project Overview

The AI Language Translation Tool allows users to enter text, select source and target languages, and translate the text through an interactive web interface.

## ✨ Features

- Supports multiple languages
- Automatic source-language detection
- Source and target language selection
- Empty input handling
- Same-language validation
- Interactive Gradio interface
- Public shareable link through Gradio

## 🧠 Technologies Used

- Python
- Deep Translator
- Google Translator
- Gradio
- Google Colab

## ⚙️ How It Works

1. The user enters text.
2. The user selects the source language or chooses Auto Detect.
3. The user selects the target language.
4. Google Translator processes the input.
5. The translated result is displayed in the Gradio interface.

## 📂 Project Files

- `CodeAlpha_Language_Translation_Tool.ipynb` — Complete implementation
- `README.md` — Project documentation

## ▶️ How to Run

1. Open the notebook in Google Colab.
2. Run the installation and import cells.
3. Run the language dictionary and translation function.
4. Run the Gradio interface cell.
5. Open the generated public link or use the inline interface.
